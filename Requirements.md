**Requirements**

* Streamlit: Used to quickly build the interactive web interface, forms, and display visualizations
* Pandas: Primary library for loading, cleaning, and manipulating the applicant data in memory.
* NumPy: Provides fast, efficient array operations, essential for complex math like EMI formulas.
* Scikit-learn: Provides data preprocessing, feature scaling, model selection, and overall workflow management.
* XGboost: The high-performance machine learning model used to predict the probability of loan default.
* SHAP: Calculates the contribution of each feature to the final prediction, generating SHAP values.
* Matplotlib: A foundational library used for generating static plots (e.g., feature distribution or simple SHAP summary plots).
* Joblib: Used to efficiently save and load the trained machine learning model file (model.pkl).
* plotly
